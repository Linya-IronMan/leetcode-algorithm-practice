# LeetCode 9. Palindrome Number

- 链接：[LeetCode Palindrome Number](https://leetcode.cn/problems/palindrome-number/)
- 难度：Easy

## 题目描述

给你一个整数 `x` ，如果 `x` 是一个回文整数，返回 `true` ；否则，返回 `false` 。

回文数是指正序（从左向右）和倒序（从右向左）读都是一样的整数。

### 示例 1：

```
输入：x = 121
输出：true
```

### 示例 2：

```
输入：x = -121
输出：false
解释：从左向右读, 为 -121 。 从右向左读, 为 121- 。因此它不是一个回文数。
```

### 示例 3：

```
输入：x = 10
输出：false
解释：从右向左读, 为 01 。因此它不是一个回文数。
```

### 提示：

- `-2^31 <= x <= 2^31 - 1`

### 进阶：

- 你能不将整数左右反转为字符串来解决这个问题吗？

## 解题思路

### 方法：反转一半的数字
为了避免数字反转后可能导致的整数溢出问题，我们可以在不将整数转换为字符串的情况下，只反转整数的后半部分，并将其与前半部分进行比较。

1. **特殊情况拦截**：
   - 负数必定不是回文数（例如 `-121` 的负号会导致反向读取不一样）。
   - 除了 `0` 以外，所有以 `0` 结尾的数字都必定不是回文数（因为首位不可能是 `0`）。
2. **反转数字**：
   - 每次将原数字 `x` 的最后一位取出来并累加到反转结果中：`reverted = reverted * 10 + x % 10`。
   - 然后将 `x` 去除最后一位：`x = x / 10`。
   - 当 `x <= reverted` 时，说明我们已经处理了数字的一半或更多，循环终止。
3. **比较结果**：
   - 如果数字是偶数长度，则 `x == reverted` 说明是回文。
   - 如果数字是奇数长度，我们可以通过 `reverted / 10` 去除处于中位的数字（因为中位数字不影响回文），然后判断 `x == reverted / 10`。
